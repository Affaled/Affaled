<p>Não sei exatamente quando comecei a programar. Talvez tenha sido quando percebi que podia dar forma a uma ideia só com lógica e paciência. Talvez tenha sido quando entendi que criar código era, de algum jeito, parecido com contar histórias. Até agora, minha jornada tem sido principalmente pelo <em>front-end</em> — <strong>HTML</strong>, <strong>CSS</strong>, <strong>JavaScript</strong>, <strong>Angular</strong>, essas coisas. Fiz <em>sites</em>, <em>landing pages</em>, experimentei frameworks, brinquei com <strong>WordPress</strong> e <strong>Liquid</strong>. Mas, sendo bem honesto, nunca me vi só como "dev front-end". Isso foi só o começo. A verdade é que estou explorando. Estudando <em>sistemas</em>, aprendendo <em>lógica</em>, tentando entender o que me move dentro da programação. Talvez sejam os <em>jogos</em>. Talvez seja o poder de <em>automatizar</em> coisas. Talvez seja só a vontade de entender como tudo se encaixa.</p>
